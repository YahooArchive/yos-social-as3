generate_nonce()method 
public static function generate_nonce(date:Date = null):String

Generates an MD5 encoded string based on the current date and a random number. Used in the OAuth request parameters as a method to prevent replay attacks by generating a string which is not likely to recur.

Parameters
date:Date (default = null) — A Date object. If null, the function will use the current date.
