Packagecom.yahoo.social.utils
Interfacepublic interface IYahooSessionStore
ImplementorsYahooSessionStore

IYahooSessionStore provides a basic interface for storing OAuth tokens using the com.yahoo.oauth package and YahooSession object. Your implementation of this interface allows you to write code to store tokens in anything from local shared objects, in the cloud or local files in Adobe AIR.

A basic implementation of this is found in YahooSessionStore, which uses a configurable local shared object to store tokens.
